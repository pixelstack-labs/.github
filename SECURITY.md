# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in PixelStack,
please **do not** open a public issue.

Instead, email us at **pixelstackcloud@gmail.com**
with the subject "Security Report".

We will respond within 48 hours and work with you
to resolve the issue as quickly as possible.

| Version | Supported |
|---------|-----------|
| main (latest) | ✅ |
